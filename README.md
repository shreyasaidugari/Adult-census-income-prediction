# Adult Income Prediction using Machine Learning
📌 Project Overview

This project is a Machine Learning application built to predict adult income levels based on various personal and professional attributes such as age, education, occupation, working hours, etc.

The goal of this project is to compare and analyze the performance of two machine learning algorithms:

K-Nearest Neighbors (KNN)

Decision Tree

## 🗂 Dataset Description

Dataset Name: Adult Income Dataset

Source: UCI Machine Learning Repository

The dataset contains both categorical and numerical features.

Target Variable: Income (e.g., <=50K or >50K)

## Technologies Used

Python

Pandas

NumPy

Scikit-learn

Streamlit (for web application)

## 🔄 Project Workflow

Data Loading & Preprocessing

Handled missing values

Encoded categorical features

Scaled numerical features

Model Building

Implemented KNN Algorithm

Implemented Decision Tree Algorithm

Model Evaluation

Compared accuracy and performance of both models

Selected the best-performing model for prediction

Deployment

Built an interactive Streamlit web app

Users can enter input values and get income prediction

## Machine Learning Algorithms Used
🔹 K-Nearest Neighbors (KNN)

Predicts income based on similarity with nearest data points

Works well with scaled numerical data

🔹 Decision Tree

Uses decision rules to split data

Easy to understand and interpret

Handles both numerical and categorical data effectively

##  Results

Both models were trained and tested successfully

Model performance was compared to choose the best one

Final model gives reliable predictions for new input data
